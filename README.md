 Wallet App Transfer

Aplicație simplă de gestionare a portofelelor și transferuri între utilizatori, creată în Python.

 Descriere

Acest proiect simulează un sistem de portofele digitale pentru trei utilizatori: Ana, Mihai și Dumitru. Fiecare utilizator are un portofel cu un sold inițial, iar aplicația permite efectuarea de transferuri între portofele și afișarea rezultatelor finale.

Funcționalități

- Crearea utilizatorilor cu sold inițial
- Transfer de bani între portofele
- Afișarea soldurilor actuale
